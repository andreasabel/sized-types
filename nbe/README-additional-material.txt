Additional material for ICFP 2017 submission #84

* icfp17-long.pdf

  The full version of the article, containing additional theorems and proofs.

* Sit-0.2017.2.26.tar.gz

  An implementation of the type checker described in the article.

  Requires a recent Haskell Platform (ghc >= 7.8, cabal).

  Run with shell commands:

  $ tar xzf Sit-0.2017.2.26.tar.gz
  $ cd Sit-0.2017.2.26
  $ make
